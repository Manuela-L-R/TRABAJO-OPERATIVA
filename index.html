📋 Descripción del Problema

🔄 Modelo Primal vs Dual

📌 MODELO PRIMAL (Decisión)

Objetivo: Minimizar total de cajeros

Variables:

x₁, x₂, x₃, x₄, x₅, x₆

= Cajeros que inician turno en cada período

Función Objetivo:

Z = x₁ + x₂ + x₃ + x₄ + x₅ + x₆

Solución Óptima:

Z* = 45 cajeros

💎 MODELO DUAL (Valoración)

Objetivo: Maximizar valor de cobertura

Variables:

y₁, y₂, y₃, y₄, y₅, y₆

= Precio sombra de demanda cada período

Función Objetivo:

W = 7y₁ + 20y₂ + 14y₃ + 20y₄ + 10y₅ + 5y₆

Solución Óptima:

W* = 45 (igual al primal ✓)

📐 Verificación Matemática

🔗 Dualidad Fuerte

Teorema: Si el problema primal tiene solución óptima finita, su dual también la tiene, y son iguales. Z* = W* 45 = 45 ✓ VERIFICADO

🔀 Holgura Complementaria

Teorema: Para soluciones óptimas (x*, y*), se cumple:

  • Si x*ᵢ > 0, entonces su restricción dual debe ser activa (=1)
  • Si restricción primal tiene holgura, su variable dual debe ser 0
