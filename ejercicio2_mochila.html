📋 Descripción del Problema

Un excursionista planea salir de campamento y quiere llevar 5 artículos consigo. Sin embargo, tiene una restricción: puede cargar como máximo 60 libras. Cada artículo tiene un peso y un valor asignado. ¿Cuál es la combinación óptima de artículos que maximiza el valor total sin exceder el peso límite?

📐 Formulación Matemática

Variables de Decisión

x_i = 1 si se lleva el artículo i x_i = 0 si no se lleva el artículo i (donde i = 1, 2, 3, 4, 5) Estas son variables BINARIAS (solo pueden ser 0 o 1)

Función Objetivo (MAXIMIZAR)

Z = 100x₁ + 60x₂ + 70x₃ + 15x₄ + 15x₅ Objetivo: Maximizar el valor total de los artículos llevados

Restricción de Peso

52x₁ + 23x₂ + 35x₃ + 15x₄ + 7x₅ ≤ 60 El peso total no puede exceder 60 libras

Clasificación del Problema

  • Tipo: Problema de Mochila 0/1 (Knapsack Problem)
  • Modelo: Programación Lineal Entera Binaria
  • Variables: 5 variables binarias (x₁ a x₅)
  • Restricciones: 1 restricción de peso + dominio binario
  • Objetivo: Maximizar valor bajo restricción de capacidad

Mejor: llevar dos artículos más pequeños con mayor valor total.
